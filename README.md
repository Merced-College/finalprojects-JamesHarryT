# Media Tracker Project for CPSC-39 Final

## Description
This is a terminal-based Java application that allows users to track and manage different types of media
such as books, movies, games, songs, and shows. Users can add, remove, edit, and view media, as well as
maintain a watch list.

## How to Run
Note: This was made in Eclipse so that's the recommended method. However, opening this on GitHub Codespaces seems to work as well
1. Open the project in a Java IDE like Eclipse
2. Run Main.java
3. Follow the terminal prompts

## Features
- Add, remove, and edit media
- View individual media (to see notes)
- View all media
- Sort media by title, rating, or type
- Watch list system (queue-based)
- Save/load data from file

## Data Structures Used
- ArrayList (stores all media)
- HashMap (fast searching by title)
- LinkedList (watch list queue)

## Algorithms Used
- Selection sort (sorting media)
- Linear Search (removing from watch list)
- HashMap search (fast searching)
