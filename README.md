# Media Tracker Project for CPSC-39 Final
This program is tracks media (games, shows, and movies) for the user. The program should be cloned to Eclipse for the best experience.
Some features are that you can add, remove, and edit media. It will all be saved and loaded to a csv file and there is also a section for 
if the user has watched or played the media. If they haven't enjoyed the media before it can be used as a watch list of sorts.
